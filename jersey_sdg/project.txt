This project's purpose is to create a dataset that can be used to train:
1. A jersey recognition model capable of telling us what a given player image crop's jersey number is. This model needs to be capable of saying whether the jersey number is visible/legible and if so, what the number is.
2. A re-identification network that given a gallery of know players/identities and a image crop of a player can tell us which player out of the gallery it is.

We have dataset in /mnt/t/data/vball/skyball that is MOT-style in that for each video, there are player tracks (annotations embedded within the json for that video). And these annotations also include the jersey number of that player.  The challenge with leveraging this data is that the jersey number is often not clearly visible if we were to take an image crop of that player on a given frame using the gt bbox.

Proposal:

Use Qwen 3.7 Plus as a synthetic labeller.
You could use a prompt such as this: "Can look at this player image and tell me what the jersey number is? if you're unsure or the digits are not visible, then say that"

Ideally you can get this to run efficiently on our system, i don't know how well Qwen 3.7 plus would work. Perhaps one of the quantized variants would work.

Sampling ...
For each video of our dataset, its a volleyball play of maybe 10-20 seconds.
The basename of the file encodes the match name and there should be a suffix in the filename like 'play1' or 'play2'.

We want a few things:

A. For re-id:
  For each match, for each player, a gallery of images of that player. Ideally the primary player centered in the image is not entirely or nearly full occluded, else the re-id network would have a difficult time.

B. For jersey identification:
  Assemble a dataset of player crops along with visible/not visible and the jersey GT. In an ideal world, we have a healthy mixture of positive/negative examples such that the jersey detector learns well.


Start by generating some trial runs ofr A and B. Then share them with me.
If things look good, we'll want to scale it up.
